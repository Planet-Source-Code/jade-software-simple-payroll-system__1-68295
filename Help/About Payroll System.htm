About Payroll System

This software is a true 32-bit Windows application designed to meet your payroll needs. The software can be able to speed-up the process of payroll computation, searching of employee, monitoring of SSS Loan and SSS Premium, PhilHealth, Pag-ibig and other deductions. The software handles all the reporting you needs and create a real-time reports for your hard copy of data. Designed to function simply and effortless with only the use of a mouse and keyboard.

The software runs in any modern operating system like Windows 9x, Windows ME, Windows 2000, Windows NT or Windows XP.
